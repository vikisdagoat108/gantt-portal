Import Tasks from Excel / CSV

For large spreadsheets, bring tasks in all at once instead of typing them by hand. Either upload a file or paste cells copied directly from Excel. Imported rows land in the grid below as unsaved (green) rows — review or fix anything, then click Save All Changes as usual.

Step 2: Task Generation & Lag Matrix (Spreadsheet Mode)

Type directly into cells like a spreadsheet. Add as many rows as you need, then click Save All Changes once. Existing tasks load automatically and can be edited in place. Columns mirror your Output Sheet: Task ID, Category, Weeks, Parent ID.

Import Dependencies from Excel / CSV

Bring your dependency block in all at once. Tasks must already be saved (with matching Task IDs) before importing — each row is matched to a saved task by its Task ID. Expected columns: Source Task ID, Target Task ID, Flag, Days.

Step 3: Dependency Matrix (Logic Links & Lag)

Define which tasks depend on which. Pick a Source (predecessor) and Target (successor) from your saved tasks, choose the Flag (FS = Finish-to-Start, SS = Start-to-Start, FF = Finish-to-Finish, SF = Start-to-Finish), and set the Days lag. Columns mirror your Output Sheet dependency block. Save your tasks first in Step 2 so they're available to select here.

Step 4: Gantt Chart

Visualizes every saved task with a Start Date and End Date as a timeline bar, colored by Progress %, with dependency arrows drawn between linked tasks. Tasks missing dates are listed separately below the chart — fill in their Start/End Date and regenerate to include them.

Click "Generate / Refresh Gantt Chart" once your tasks have dates.
